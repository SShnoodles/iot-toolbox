# Iot Debugging Tool

This is a cross-platform debugging tool for serial communication and more...

Features

- [x] Serial port debugging
- [ ] Future support for Modbus debugging

## Build

### MacOS or Linux

```bash
cargo build --release
```

### Windows

```bash
rustup target add x86_64-pc-windows-msvc
cargo build --release
```
